## Run Application

set env variables:
- database url
- backend url

### Client

go to `client`

run:

```
npm install
npm start
```

client will open at `localhost:3000`

### Server

go to `server`

run:

```
pip install -r requirements.txt
uvicorn main:app --reload
```

server will open at `localhost:8000`
